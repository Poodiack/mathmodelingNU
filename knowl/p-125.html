We're going to start with a very easy subroutine, which would be part of the programming of a vending machine or a cash register. It is going to reply with the dollar value for some number of quarters, dimes, nickels, and pennies, which of course have respective values $0.25, $0.10, $0.05, and $0.01. This is a simple task, so that we can concentrate on the programming rather than the underlying mathematics.
